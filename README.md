# SolarTest

### Start project
* Run `npm run build` to install dependencies.
* Run `npm run deploy` to deploy github-pages

### Start dev
* Run `npm i` to install dependencies.
* Run `npm start` for a dev server. Navigate to `http://localhost:4200/`.
